# Gallery-Portfolio

## 部署步骤

1. 将 `.env_template` 改名为 `.env`，并修改其中的配置
2. 使用以下方法部署：

```console
npm init -y
npm install @aws-sdk/client-s3 sharp dotenv express
node server.js
```
